ONCE ON QUESTION PAGE
USER CLICKS RADIO BUTTON 
AND THEN CLICKS THE SUBMIT
BUTTON FOR ANSWER FEEDBACK

------------------->     IF THE ANSWER IS CORRECT
                         USER IS DIRECTED TO THE 
                         CORRECT ANSWER PAGE AND 
                         PROVIDED THE CURRENT SCORE!
                         USER THEN CLICKS BUTTON TO 
                         HEAD BACK TO QUESTION PAGE FOR
                         NEXT QUESTION

------------------->     IF THE ANSWER IS INCORRECT
                         USER IS DIRECTED TO THE 
                         INCORRECT ANSWER PAGE AND 
                         PROVIDED THE CURRENT SCORE!
                         USER THEN CLICKS BUTTON TO 
                         HEAD BACK TO QUESTION PAGE FOR
                         NEXT QUESTION

-------------------->

ONCE THE USER HAS ANSWERED 
ALL THE QUESTIONS, UPON CLICKING
THE SUBMIT BUTTON FOR THE FINAL 
QUESTION PAGE, USER WILL RECEIVE 
FEEDBACK ON THEIR QUIZ RESULTS
AND THEN HAVE AN OPPORTUNITY TO 
RETAKE THE QUIZ BY CLICKING THE 
START AGAIN BUTTON
